Projects

K-Means Clustering Tool

Full-stack web application that utilizes the K-Means machine learning algorithm to segment images based on dominant colors. The application allows users to upload an image, enter a desired value of K for the number of clusters, view and download the segmented image. The application relies on HTML, CSS, and JavaScript for the frontend while the backend is built using Python and Flask. The K-Means algorithm was built from scratch in Python using NumPy and OpenCV while Flask was used for building the API to connect with the frontend.

Convolutional Neural Networks for Image Style Transformation

Developed a convolutional neural network model that can successfully apply the style of an image to the semantic style of another. The model was implemented in Python using TensorFlow, OpenCV, and a pretrained VGG-19 model. This project was inspired by the research published by Gatys, et al. on the same topic.

Twitter Sentiment Analysis using Natural Language Processing.

Performed emotional sentiment analysis on 400k text samples from Twitter. Performed various NLP techniques, such as tokenization, stop word removal, etc. Trained multiple Machine Learning models (including Logistic Regression, Naive Bayes, and Multilayer Perceptron) using Scikit-learn, and achieved an average accuracy of 87%.
